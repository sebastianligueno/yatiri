# Roadmap

## Near term

- improve web search ranking and source typing
- enrich `teach`, `design`, `quant`, and `qual` modes
- add explicit citation-aware answer formatting
- expose clearer runtime diagnostics for DeepSeek and Ollama failures
- add first-class MCP integration for academic retrieval and verification
- add explicit search workflows for SciELO and other open Spanish-language academic bases

## Medium term

- optional bibliography connectors
- prompt templates for class plans, rubrics, and research designs
- verification workflows for manuscripts and arguments
- better attached-context indexing for large folders

## Longer term

- plugin architecture for discipline-specific workflows
- richer terminal UI
- configurable memory profiles
- optional export to Markdown, Quarto, and teaching materials
